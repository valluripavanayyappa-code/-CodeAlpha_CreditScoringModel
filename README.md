# 💳 Credit Scoring Model

A Machine Learning project developed as part of the **CodeAlpha Machine Learning Internship**.

This project predicts whether a loan applicant is creditworthy based on financial and personal information using supervised machine learning algorithms.

---

## 📌 Project Overview

Credit scoring is an essential process used by banks and financial institutions to determine whether a customer is eligible for a loan. This project builds a classification model that predicts an applicant's creditworthiness using historical financial data.

---

## 🎯 Objective

- Predict whether a customer is likely to repay a loan.
- Compare different Machine Learning classification algorithms.
- Improve prediction accuracy using data preprocessing and feature engineering.

---

## 📂 Dataset

The dataset contains customer financial information such as:

- Age
- Annual Income
- Employment Status
- Loan Amount
- Credit History
- Existing Debt
- Number of Credit Accounts
- Payment History
- Credit Score Category

> **Note:** The dataset used is for educational purposes.
